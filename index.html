Ferramentas Utilizadas

(Voltar)

O ambiente de desenvolvimento a ser utilizado durante o curso será o Visual Studio Code, que pode ser baixado aqui. O Visual Studio Code é uma IDE (Integrated Development Environment) ou simplesmente Ambiente de Desenvolvimento Integrado, em português. Uma IDE é um software que combina diversas ferramentas em uma interface gráfica, e serve para auxiliar na escrita de códigos. O VS Code é o editor de códigos desenvolvido pela Microsoft.

Utilizando o inspetor de elementos

(Voltar)

O inspetor de elementos é uma ferramenta oferecida pelo próprio navegador, e que como o próprio nome diz serve p/ inspecionar os elementos do documento HTML. Além de inspecionar, essa ferramenta pode ser usada para fazer alterações temporárias no documento e ver os resultados em tempo real, sem precisar mexer no documento original. Isso pode ser bastante útil para testar uma alteração ou diagnosticar um problmea. Ou, ainda, quando você encontra um site com um recurso que você gosta e fica curioso para saber como foi implementado.

Estrutura básica do HTML

(Voltar)

A estrutura básica do HTML compreende as tags html, head e body. A tag html indica para o navagador o tipo daquele documento. A tag head é onde vai ser colocado o cabeçalho da página, com as tags meta, title, style, entre outras. E na tag body, como o próprio nome já sugere, vai o corpo da página, que também poderá ser dividido em seções para melhor entendimento futuro do código.

Falando sobre Tags

(Voltar)

As tags são os elementos do HTML. É através delas que o navegador identifica e interpreta os diversos tipos de textos que compõem o documento. São escritas sempre entre os sinais < (menor que) e > (maior que), e podem ter o que é chamado de tag de fechamento ou não. A tag de fechamento contém uma / (barra) após o sinal < (menor que). Existem ainda as tags que são chamadas de semânticas, e que servem apenas para organizar e facilitar a consulta do código. Alguns exemplos de tags semânticas são header, main e footer .

Atributos Básicos

(Voltar)

Atributos são palavras específicas que são usadas dentro das tags html a fim de controlar seu comportamento. Através de atributos é possível alterar cores, tamanhos, fundos, etc.. Existem atributos que são globais, ou seja, podem ser usados em todas as tags, e exitem atributos específicos para determinados tipos de tags, como src, href, disable, etc...

Executando uma página HTML

(Voltar)

Normalmente o arquivo com a extensão .html ou .htm já está associado a um determinado navegador, que é escolhido no sistema como navegador padrão para exexutar tais arquivos. E nesse caso basta dar dois cliques sobre o arquivo ou um clique com o botão direito e ir na opção abrir.

Textos

(Voltar)

Existem inúmeras tags destinadas a formatação de textos no HTML, sendo as mais comuns a p, que é de parágrafo, as h1 ao h6, que são usadas basicamente para dar destaque a títulos, e as que alteram algum comportamento do texto, como negrito, itálico, underline, entre muitas outras que poderão ser abordadas em outros artigos.

Listas Ordenadas e Não Ordenadas

(Voltar)

Outro elemento muito usado no HTML são as listas. Existem basicamente as listas ordenadas, representadas pela tag ol (ordered list), que são automaticamente incrementadas com um número ordinal à medida que são inseridas no documento, e as não ordenadas, representadas pela tag u (unordered list), que adicionam apenas um elemento gráfico em cada linha, como uma bolinha, um quadrado ou algum outro símbolo. Para que sejam exibidas as listas ainda contam com outra tag, a li, que é colocada dentro das tags citadas acima, e essa que é a responsável pela lista propriamente dita.

Lista Ordenada

  1. Item1
  2. Item2
  3. Item3

Lista não Ordenada

  • Item1
  • Item2
  • Item3

Links

(Voltar)

Já para a criação de links dentro do documento HTML usamos basicamente a tag a, seguida do seu atributo href, que significa Hypertext Reference, e é responsável pela informação do destino daquele link, que pode ser um endereço externo ou ainda alguma seção da própria página. Através do atributo target é possível escolher se o link será aberto na mesma aba ou em uma aba separada do navegador.

Mais algumas Tags

(Voltar)

Além das tags já citadas acima iremos falar de mais algumas, que são:

  • font - está assim riscada justamente para indicar que não é mais usada no html5.
  • del - serve para fazer o efeito acima, de riscar uma palavra ou parte do texto.
  • mark - usada para destacar uma parte do texto: assim..
  • sub - adiciona um texto meia linha abaixo da linha atual. Um de seus usos mais comuns é em fórmulas químicas, como em: H2O.
  • sup - adiciona um texto meia linha acima da linha atual, comumente usada para numeros exponenciais e citações de rodapé. Ex: 32, Twain1
  • blockquote - serve para referenciar uma citação em um texto. O texto dentro dessa tag recebe um parágrafo à parte, com uma identação, como no exemplo abaixo:
    The blockquote HTML element indicates that the enclosed text is an extended quotation. Usually, this is rendered visually by indentation (see Notes for how to change it). A URL for the source of the quotation may be given using the cite attribute, while a text representation of the source can be given using the element.
  • abbr - define uma abreviação, como a seguinte. A World Wide Web (WWW) foi criada por Tim Burners Lee. O atributo title é usado para mostrar uma pequena descrição quando o mouse for passado por cima da abreviação.
